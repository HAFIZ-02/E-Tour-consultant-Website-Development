# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-05-02

### Added - Frontend
- Next.js 14 application with App Router
- Landing page with hero section and category filter
- Tour packages browsing with search and filter
- Transport booking system (flights, buses, car rentals)
- Booking form with validation
- Checkout confirmation page
- Responsive design with Tailwind CSS
- Toast notifications for user feedback
- Receipt download functionality
- Dummy data with 10 tour packages and 7 transport options
- Navbar and Footer components
- TypeScript support
- API client setup with Axios

### Planned - Backend

- Booking management
- Email system with Nodemailer
- Admin endpoints
- Database seeding

### Planned - Features
